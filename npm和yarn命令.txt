npm install -g cnpm --registry=https://registry.npm.taobao.org 下载cnpm
nvm install latest 安装最新版本的node
nvm uninstall v12.7.0 卸载 12.7.0 版本的 Node
nvm use 13.6.0 使用node13.6.0 nvm中必须指定使用哪个版本的node
nvm ls available 查看node发行版本
nvm list 查看已下载node版本   LTS 最新稳定版 current 最新版nvm
npm 是依附于 node.js 的，我们可以去它的官网 https://nodejs.org/en/download/ 下载安装 node.js。下载好 node.js, npm 也就有了，
cnpm install webpack@3.6.0 -g  安装3.6.0的webpack   -g是全局安装的意思
webpack -version/webpack -v 查看webpack版本
cnpm install vue@2.6.11 -g /cnpm install -g vue@2.6.11   全局安装2.6.11版本的vue
cnpm install -g @vue/cli
npm -v   查看npm版本
npm version 查看npm和npm包版本
npm search 包名   搜索包含包名的包
npm install/i 包名 -g  yarn global add 包名   全局模式安装包
npm install/i 包名       yarn add  包名     在当前目录安装包
npm init  生成一个package.json
npm remove/r 包名  yarn remove 包名  卸载包
cnpm install/i    yarn  根据package.json/yarn.lock安装依赖项包 
yarn create react-app 项目名 创建react项目
